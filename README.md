# Histopathology Viewer

### Setup
- Install MySQL, and create credentials as mentioned below
```
SQL_HOST = "localhost"
SQL_USER = "Histuser"
SQL_PASSWORD = "password"
SQL_DATABASE = "Histdata"
```
- Run `create_schema.sql` from `server/db_setup`
